# Binary SearchTree

정점 == 노드 == 원소 <br/>
루트 == 최상단의 노드 <br/>
간선 = 정점을 연결하는 선 <br/>
부모정점, 자식정점 <br/>
서브트리 <br/>
트리의 높이

### 이진 트리

<blockquote>
각 노드의 자식이 2개 이하인 트리 
</blockquote>

### 이진 검색 트리

<blockquote>
왼쪽 서브트리의 모든 값은 부모의 값보다 작고 오른쪽 서브트리의 모든 값은 부모의 값보다 큰 이진트리
</blockquote>

시간복잡도 => 삽입, 삭제, 검색, 수정 _O(logN)_
원소가 크기 순으로 정렬되어 있음
